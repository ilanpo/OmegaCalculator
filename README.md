# OmegaCalculator
Calculator for omega written in python 3.10 with tests in pytest 6.2

Powerful calculator that can calculate using all sorts of operators 

Operators: + Addition, - Subtraction and Negation, * Multiplication, / Division,
^ Power, % Modulo, $ Maximum, & Minimum, @ Average, ~ Negate, 
! Factorial (has to be whole and positive), # Sum of digits.

Usage instructions: type exit to end, otherwise input any mathematical equation and get the answer

Functions like a compiler with a lexer parser and solver.
